Sobre esta Aplicação

Este programa foi desenvolvido com o auxílio de Inteligência Artificial (IA) e será utilizado como produto educacional em uma dissertação de mestrado do PROFMAT.

Como Usar

  • Cole o código: Insira seu script Manim na caixa de texto.
  • Qualidade: Escolha a qualidade da animação. Limitado para fins de teste no servidor web. Para vídeos em 1080p, utilize a versão Desktop.
  • Renderizar: Executa o código e abre o player de vídeo.
  • Para baixar: Após renderizar, utilize os 3 pontinhos no canto inferior direito do vídeo..
